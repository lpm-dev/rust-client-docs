---
title: lpm remove
description: "Reverse an lpm add — remove a source-delivered package from your project."
---

```bash
lpm remove <package>   # alias: lpm rm
```

Removes the source files that `lpm add` previously copied into your project, plus any tracking entry LPM kept for it.

This is the **reverse of [`lpm add`](/docs/packages/add)**. To remove a runtime dependency from `package.json` and `node_modules`, use [`lpm uninstall`](/docs/packages/uninstall) instead.

## Examples

```bash
lpm remove @lpm.dev/owner.ui-kit
lpm rm lodash.merge
```

- [`lpm uninstall`](/docs/packages/uninstall) — remove a runtime dependency
