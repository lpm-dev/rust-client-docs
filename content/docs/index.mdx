Welcome. LPM is the fast, all-in-one toolkit for modern software — written in Rust, one binary.

## Install

```bash
npm install -g @lpm-registry/cli
```

```bash
brew tap lpm-dev/lpm && brew install lpm
```

```bash
curl -fsSL https://lpm.dev/install.sh | sh
```

```bash
cargo install --git https://github.com/lpm-dev/rust-client lpm-cli
```
